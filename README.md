# Predictive Analytics Using Historical Data

## 📌 Project Overview
This project predicts future sales trends using historical sales data with the help of Machine Learning. A Linear Regression model is trained on past monthly sales data to forecast future sales.

## 🎯 Objective
- Analyze historical sales data.
- Build a predictive model using Linear Regression.
- Forecast future sales trends.
- Visualize actual and predicted values.

## 🛠️ Technologies Used
- Python
- Pandas
- Matplotlib
- Scikit-learn
- Visual Studio Code

## 📂 Project Structure
```
Predictive_Analytics_Project/
│── predict.py
│── sales_data.csv
│── README.md
```

## 📊 Dataset
The dataset contains monthly sales records for 10 months.

| Month | Sales |
|------:|------:|
| 1 | 120 |
| 2 | 135 |
| 3 | 150 |
| 4 | 160 |
| 5 | 175 |
| 6 | 190 |
| 7 | 210 |
| 8 | 225 |
| 9 | 240 |
| 10 | 260 |

## 🚀 How to Run
1. Install the required libraries:
   ```
   pip install pandas matplotlib scikit-learn
   ```
2. Place `sales_data.csv` in the project folder.
3. Run the program:
   ```
   python predict.py
   ```

## 📈 Output
- Predicts sales for future months.
- Displays predicted values in the terminal.
- Generates a graph showing actual and predicted sales.

## 📌 Results
The model successfully predicts future sales using historical data and demonstrates the application of Linear Regression for forecasting.
